Guessing Game
-------------

This code allows a user or computer 5 tries to guess a number between 1-100.
Incorrect guesses will be noted as to high or to low

The computer uses a binary search algorithm to formulate it's guesses

To Use
------
To run the user guessing game input 'ruby game.rb' into the command line

To run the computer guessing game input 'ruby BinaryAlgGame.rb' into the command line
